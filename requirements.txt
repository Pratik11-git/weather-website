For successfully run this project you need to follow following straps:
1)create new folder, inside unzip this project 
2)create virtual environment: pyhton -m venv env_name
3)activate virtual environment: env_name\Scripts\activate.bat
4)install django: pip install django
5)install requests package(for sending http requests): pip install requests 
(go to inside the project directory 'weather_website' using cd command)
6)run sever : python manage.py runserver
